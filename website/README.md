# 🎬 TikTok Downloader Website

موقع ويب جميل وحديث لتحميل فيديوهات تيك توك بدون علامة مائية!

## ✨ المميزات

- 📥 تحميل فيديوهات TikTok بدون علامة مائية
- 👤 عرض معلومات الحسابات والإحصائيات
- 🎨 تصميم حديث مع تدرجات ألوان جميلة
- ✨ تأثيرات زجاجية (Glassmorphism)
- 🌊 رسوم متحركة سلسة
- 🌙 وضع داكن عصري
- 📱 متجاوب مع جميع الأجهزات
- 🇸🇦 دعم كامل للغة العربية (RTL)

## 🚀 التثبيت والتشغيل

### المتطلبات
- Node.js (الإصدار 14 أو أحدث)
- npm أو yarn

### خطوات التشغيل

1. **انتقل إلى مجلد الموقع:**
```bash
cd website
```

2. **تثبيت المكتبات:**
```bash
npm install
```

3. **تشغيل الموقع:**
```bash
npm start
```

4. **افتح المتصفح:**
افتح المتصفح وانتقل إلى: `http://localhost:3000`

## 📖 كيفية الاستخدام

### تحميل فيديو:
1. افتح تطبيق تيك توك واختر الفيديو المطلوب
2. اضغط على زر "مشاركة" ثم "نسخ الرابط"
3. الصق الرابط في الموقع
4. اضغط على "تحميل"
5. احفظ الفيديو على جهازك!

### معلومات الحساب:
1. انتقل إلى تبويب "معلومات الحساب"
2. أدخل اسم المستخدم (مع أو بدون @)
3. اضغط على "بحث"
4. شاهد جميع معلومات الحساب والإحصائيات!

## 🛠️ التقنيات المستخدمة

### Backend:
- **Express.js** - خادم الويب
- **@tobyg74/tiktok-api-dl** - مكتبة تحميل TikTok
- **Axios** - طلبات HTTP
- **CORS** - للسماح بالطلبات من الواجهة الأمامية

### Frontend:
- **HTML5** - البنية الأساسية
- **CSS3** - التصميم والرسوم المتحركة
- **JavaScript** - التفاعل والوظائف
- **Google Fonts (Tajawal)** - خط عربي جميل
